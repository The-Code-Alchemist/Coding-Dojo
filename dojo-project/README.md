# 🧪 Coding Dojo — Dojo Project

A structured learning environment for mastering Java, DevOps, and software engineering through progressive belt levels.

---

## 🧠 Overview

This project is part of the Coding Dojo.

It evolves step by step as you progress through the belts — from basic setup to full system design.

---

## 🚀 Running the Dojo

Use the provided script to start the application.

### All Platforms (Linux / macOS / Windows via Git Bash)

```bash
bash scripts/run-dojo.sh
```

> “One command. One workflow. No confusion.”

---

## 🌐 API Endpoints

## 🧠 Belt Mindsets

⚪ **White Belt**
“I learn how things work.”

🟡 **Yellow Belt**
“I can build something that works.”

🟠 **Orange Belt**
“I understand the system behind what I build.”

🟢 **Green Belt**
“I write code that others can understand, extend, and maintain.”

🔵 **Blue Belt**
“I design systems, not just applications.”

🟣 **Purple Belt**
“I automate, standardize, and optimize the development process.”

🟤 **Brown Belt**
“I think in trade-offs, scalability, and long-term impact.”

⚫ **Black Belt**
“I build, lead, and evolve systems and teams.”

🏆 **Master / Founder (Dan level)**
“I create systems that create value beyond myself.”

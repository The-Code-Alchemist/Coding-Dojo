# ⚪ White Belt — Survival & Setup

![White Belt Setup](assets/white-belt-setup.png)

---

## 🧠 Overview

Welcome to the White Belt.

This is the beginning of your journey in the Coding Dojo.

At this stage, your goal is not to build complex systems, but to **prepare your environment, understand your tools, and take your first steps as a developer**.

You will learn how to set up your development environment and run your first Java application — without relying on magic.

---

## 🎯 Objectives

After completing the White Belt, you will be able to:

* Set up a complete development environment
* Use the terminal for basic operations
* Compile and run a Java application via CLI
* Use Git for version control
* Push your code to GitHub
* Understand the difference between build and run

---

## 🧰 Required Setup

You will install and configure:

* Java (LTS version, e.g. 21)
* IntelliJ IDEA
* Git
* GitHub account

Recommended environment:

* OS: Ubuntu LTS (preferred) or Windows with WSL2
* RAM: 16 GB (minimum 8 GB)
* SSD storage

---

## 🖥️ Your Workspace

A clean and focused workspace is essential.

* A stable desk setup
* Minimal distractions
* Proper ergonomics
* Terminal access ready at all times

> “Your environment shapes your discipline.”

## 📚 Topics Covered

* Development environment setup
* Terminal basics (navigation, commands)
* Java basics (main method, execution)
* Maven basics (build lifecycle)
* Git fundamentals (clone, add, commit, push)
* GitHub usage

## 🧪 Assignments

---

### Assignment 1 — Environment Setup

* Install Java, IntelliJ, and Git
* Create a GitHub account
* Verify installations

---

### Assignment 2 — First Java Application

Create your application inside:

```
/dojo-project/
```

```java
package com.codealchemists.dojo;

public class DojoApplication {
    public static void main(String[] args) {
        System.out.println("Welcome to the Coding Dojo");
    }
}
```

---

### Assignment 3 — Build Your Project

Run:

```bash
mvn clean install
```

---

### Assignment 4 — Run via CLI (NO IDE)

Run your application using:

```bash
java -cp target/classes com.codealchemists.dojo.DojoApplication
```

---

### Assignment 5 — Git Workflow

* Initialize Git (if needed)
* Add your files
* Commit your changes
* Push to GitHub

---

### Assignment 6 — Project README

Update `/dojo-project/README.md`:

* What does your app do?
* How to build it?
* How to run it?

---

## ⚔️ Trial of Mastery — White Belt

To earn your White Belt, you must:

* Successfully set up your development environment
* Build your project using Maven
* Run your application via CLI (not IntelliJ)
* Use Git to commit and push your code
* Explain the difference between build and run
